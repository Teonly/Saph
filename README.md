# Saphyra / Saph

## Project Overview / Visão Geral (EN/PT)

**EN:**  
Saphyra is an educational platform designed to make learning more structured, accessible and engaging for students. The app provides guided lessons, interactive exercises, progression tracking, and a built-in AI tutor capable of helping students understand school subjects in a clear and personalized way. Visual design, mascots and interface style are still open for future definition.

**PT:**  
Saphyra é uma plataforma educacional projetada para tornar o aprendizado mais estruturado, acessível e envolvente para estudantes. O aplicativo oferece lições guiadas, exercícios interativos, acompanhamento de progresso e um tutor de IA integrado capaz de ajudar os alunos a compreender matérias escolares de forma clara e personalizada. O design visual, mascotes e estilo da interface ainda estão em aberto para definição futura.

## AI Tutor System / Sistema de Tutor IA (EN/PT)

**EN:**  
Saphyra includes an integrated AI tutor designed to help students learn efficiently. Unlike typical assistants that rely on general internet searches, this AI prioritizes a curated internal database built from trusted school textbooks and academic references. Only when necessary does it supplement answers with external information. This ensures accurate, consistent and didactically oriented explanations.

**PT:**  
O Saphyra inclui um tutor de IA integrado projetado para ajudar o aluno a aprender de forma eficiente. Ao contrário de assistentes comuns que apenas pesquisam na internet, essa IA prioriza um banco de dados interno composto por livros didáticos confiáveis e referências acadêmicas selecionadas. Somente quando necessário ela complementa respostas com fontes externas. Isso garante explicações precisas, consistentes e orientadas ao aprendizado.
