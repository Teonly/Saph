# Roadmap – Saphyra

## 1. Planejamento Inicial
- Definir módulos escolares principais
- Criar estrutura básica do app
- Definir modelo do tutor IA

## 2. Fase de Design
- Escolher paleta de cores
- Definir mascote
- Criar wireframes simples

## 3. Primeira Versão Funcional (MVP)
- Criar módulo de Matemática
- Criar sistema de exercícios simples
- Implementar protótipo do tutor IA

## 4. Expansão
- Adicionar XP, streaks e conquistas
- Criar módulo de Ciências
- Criar módulo de Linguagens
